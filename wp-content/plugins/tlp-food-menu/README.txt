== Installation ==

1. Add plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create food menu.
4. Add shortCode to display the food item.

= Requirements =

* **WordPress version:** >= 4
* **PHP version:** >= 5.2.4
